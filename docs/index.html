🚀 快速开始

1

安装

git clone https://github.com/johboby/deap--qd--nas.git
cd deap--qd--nas
pip install -r requirements.txt
2

验证

python examples/basic_usage.py
3

使用

from src.nas import create_default_qd_nas

optimizer = create_default_qd_nas()
archive, pareto = optimizer.optimize(
    problem_func, n_iterations=100
)

❓ 常见问题

Q: 需要GPU吗？

A: 不需要。框架可以在CPU上运行，GPU是可选的，会显著加速计算。

Q: 支持哪些数据集？

A: 内置支持MNIST、CIFAR-10/100、ImageNet等，也支持自定义数据集。

Q: 优化需要多长时间？

A: 取决于问题规模。CIFAR-10通常需要2-8小时（CPU）。

Q: 有商业使用许可吗？

A: MIT许可证允许商业使用。请查看LICENSE文件。
